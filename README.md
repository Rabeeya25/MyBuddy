MyBuddy is a simple and interactive web-based chatbot designed to provide emotional support and motivation through friendly conversations. Built using HTML, CSS, and JavaScript, it allows users to share their feelings and receive thoughtful, mood-based responses. Whether a user feels happy, sad, anxious, or stressed, MyBuddy responds with comforting and encouraging messages to uplift their mood. With its clean interface, soothing design, and empathetic tone, this project demonstrates how technology can create meaningful, positive interactions for everyday emotional well-being.
